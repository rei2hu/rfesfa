Instructions

  1. Connect your Discord account
  2. Wait for servers to get loaded
  3. Click a server icon to configure the server
  4. Drag and drop purple bars to configure channels
  5. Paste in a webhook url and select what events you want to be updated with
  6. Press save in the bottom right to save a guild's settings

Notifications: The bot will keep track of any roles in your server whose name starts with arch- or wf-. The other part of the role name is what the notification is for. For example, the role arch-forma will be mentioned if something with the word forma pops up. If an message sent by webhook contains the notification part of the name, it should be pinged.

Servers only show up if you have administrator/manage server permissions/are owner and the bot is also in the server. If you need any more help, you can join the support server through the Discord icon on the top right.

You can use this button to clear the cached information. Try to use this only if guilds/ channels that should show up are not showing up.
